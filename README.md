# Teste de perfil

## Computador
Nessa pasta, há a versão inicial do projeto: um computador de perfis comportamentais.

### Objetivo
O programa procura facilitar a computação das respostas à "avaliação_de_perfil.pdf", feita e aplicada pelo meu amigo Rafael. Originalmente, as respostas foram registradas em mensagens de WhatsApp, por isso o programa em "texte_de perfil_comportamental.py" computa as respostas coladas ao terminal.

Visto que os respondentes já haviam registrado as suas escolhas, o objetivo do programa não era fazer um "quiz", mas apenas ler e calcular os perfis disponíveis. O relatório final pode ser encontrado no arquivo "relatorio_perfis.txt" .

### Funcionalidades
- Teste: dá acesso à avaliação, permitindo que diferentes respondentes possam ser registrados.
- Visualização Geral: mostra todas os resultados de diferentes respondentes.
- Salvamento: gera um arquivo .txt com o relatório geral das respostas.
- Exclusão: permite excluir registros de acordo com o nome dos respondentes.
- Finalização: finaliza a execução do programa.

## Quiz
Pensando em tornar a ideia inicial mais atrativa aos respondentes, iniciei os testes do programa com uma interface gráfica, utilizando a biblioteca CustomTkinter.

### Objetivo
A avaliação inicial foi bastante útil em um contexto específico, mas era pouco amigável aos respondentes, devido à necessidade de interação com o terminal. Dessa forma, decidi criar uma versão quiz, com interface gráfica e disposição semelhante a um formulário online.

### Funcionalidades
- Teste: dá acesso à avaliação, permitindo que diferentes respondentes possam ser registrados.
- Visualização: mostra o perfil dominante do respondente.

## Observações:
- A versão quiz do programa ainda se encontra em estágio inicial. Então pretendo melhorá-la, evitando erros de cálculo e implementando novas funcionalidade como a de gerar relatórios, ou registrar respostas de diferentes respondentes em memória.
