# Teste de perfil
## Programa para computar respostas de um teste comportamental
O arquivo "teste de perfil comportamental.py" é um programa em python que computa as respostas dadas para uma avaliação que o meu amigo, rafael, escreveu.

## Objetivo
A avaliação original, disponível no arquivo "Avaliação de perfil editado.pdf" visa entender o comportamento dos respondentes por meio de perguntas que simulam situações reais.

O programa, por sua vez, procura computar respostas mistas, que foram dadas por arquivos de texto (txt), e responder o perfil comportamental a que o respondente mais se adequa.
